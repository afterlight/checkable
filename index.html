Checkable

A small custom checkbox/radio script.

It's super basic - wraps the radio buttons and checkboxes with an additional label.

This label can then be styled as required, it will be toggled with a .checked class when the checkbox changes state.

Usage

I'm going to assume you'll do the basics, including jQuery and the plugin JavaScript and CSS files.

To implement, simply call .checkable() on a jQuery object that will serve as the context. Let's say you want to change all the checkboxes on your site that are inside a form with the id "mainForm":

$("#mainForm").checkable();

That's it. You're done with the JavaScript, you can now worry about making your checkboxes look good!

The script will wrap a label with a class of .checkable-field around the checkbox, additionally .checkable-radio and .checkable-checkbox will be added to radio buttons and checkboxes respectively.
